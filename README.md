## 🧠 About This Project

This is a **production-grade portfolio application** built using modern frontend architecture principles.

It is designed to:

* Showcase real-world engineering work
* Highlight performance and UX decisions
* Demonstrate scalable component design

Unlike basic portfolios, this project focuses on **how things are built**, not just how they look.

---

## ✨ Key Features

* ⚡ Smooth scroll with active section tracking (Intersection Observer)
* 📊 Animated scroll progress bar with optimized rendering
* 🧩 Reusable, modular component architecture
* 🎯 Centralized data management via `data.json`
* 📱 Fully responsive layout across devices
* 🎨 Tailwind CSS with design tokens and consistency
* 🔗 Clean navigation with URL state syncing
* 📈 Integrated analytics via Vercel

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Umangip25/My-Portfolio.git
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## 🧩 Architecture Overview

```text
UI Components → Hooks (logic) → Data Layer (data.json)
```

* **Components** handle UI
* **Hooks** manage behavior (scroll tracking, state)
* **Data layer** keeps content centralized and scalable
